Introduction to Binary System

What is Binary?

Binary is a number system that uses only two digits: 0 and 1. Unlike the decimal system we use every day (which has 10 digits: 0-9), binary is a base-2 system that forms the foundation of all modern computing.

Why Do We Use Binary in Computing?

Computers use binary because digital circuits can easily represent two states:

  • 1 = ON (high voltage, true)
  • 0 = OFF (low voltage, false)

This makes binary perfect for electronic systems where switches, transistors, and logic gates work with two distinct states.

Binary Representation

How Binary Digits (Bits) Work

A bit (binary digit) is the smallest unit of data in computing. Each bit can be either 0 or 1. Multiple bits combine to represent larger numbers and data.

  • Byte = 8 bits
  • Nibble = 4 bits
  • Word = typically 16, 32, or 64 bits

Weighted vs Non-Weighted Codes

Weighted Codes

In weighted codes, each bit position has a specific weight or value. The value of the code is the sum of the weights of the 1 bits.

Examples of Weighted Codes:

1. Pure Binary (8-4-2-1)

Weights: 8, 4, 2, 1

Example: 1011 = 1√ó8 + 0√ó4 + 1√ó2 + 1√ó1 = 11

2. BCD (8-4-2-1)

Same weights, but each decimal digit is encoded separately

3. 2421 Code

Weights: 2, 4, 2, 1

Example: 0111 = 0√ó2 + 1√ó4 + 1√ó2 + 1√ó1 = 7

Non-Weighted Codes

In non-weighted codes, bit positions do not have fixed weights. The value cannot be determined by simple addition.

Examples of Non-Weighted Codes:

1. Excess-3 Code

BCD + 3 for each digit

2. Gray Code

Only one bit changes between consecutive values

Gray Code (Reflected Binary Code)

What Makes Gray Code Special?

Gray Code is a binary numeral system where only one bit changes between consecutive values. This unique property makes it extremely useful in reducing errors during transitions.

How to Convert Binary to Gray Code

Rule:

  • Keep the leftmost bit (MSB) the same
  • For each remaining bit, XOR it with the bit to its left

Example: Binary 1011 ‚Üí Gray Code

1st bit: 1 (stays same)

2nd bit: 1 XOR 0 = 1

3rd bit: 0 XOR 1 = 1

4th bit: 1 XOR 1 = 0

Result: 1110

Applications of Gray Code

Gray Code is used in situations where errors must be minimized:

  • Digital Communications: Reduces errors in signal transmission
  • Rotary Encoders: Tracks position in motors and sensors
  • Karnaugh Maps: Used in digital logic design
  • Analog-to-Digital Converters: Minimizes conversion errors
  • Genetic Algorithms: Reduces mutation errors in AI

Real-World Applications of Binary Codes

1. Computer Memory and Storage

Every file, photo, video, and document on your computer is stored as binary code. Storage capacity is measured in bytes:

  • 1 Byte = 8 bits
  • 1 Kilobyte (KB) = 1,024 bytes
  • 1 Megabyte (MB) = 1,024 KB
  • 1 Gigabyte (GB) = 1,024 MB
  • 1 Terabyte (TB) = 1,024 GB

2. Digital Images

Images are made of pixels, and each pixel's color is represented in binary:

Example: RGB Color (Red, Green, Blue)

Red = (255, 0, 0) = (11111111, 00000000, 00000000)

White = (255, 255, 255) = All bits ON

Black = (0, 0, 0) = All bits OFF

3. QR Codes and Barcodes

Barcodes: Black and white stripes represent binary data

QR Codes: 2D patterns that encode text, URLs, or data in binary format

These codes use error correction techniques (similar to what we'll learn in topics 6.3-6.5)

4. Text and Communication

Every message you send, email you write, or web page you visit uses binary codes:

  • Text messages use ASCII or Unicode
  • Web pages are transmitted in binary
  • Video calls convert audio and video to binary for transmission

5. Programming and Software

All software is ultimately compiled into machine code - binary instructions that the CPU can execute directly.

Programming languages like Python, Java, and C++ are eventually converted to binary for the computer to understand.

6. Digital Music and Audio

Sound waves are converted to binary through a process called sampling:

  • MP3 files compress audio data in binary format
  • CD quality audio uses 16-bit binary samples
  • Streaming services send binary audio data over the internet
